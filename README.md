# Movers

# Employee Management, Payroll, and Benefits System with AI Chatbot and Predictive Analytics

[GitHub Repository](https://github.com/jtorcodes/Movers/)

## Overview
This is a full-stack **Laravel application** designed to manage employees, payroll, and benefits efficiently. The system features **secure APIs**, **real-time AI chatbot support**, and **AI-powered payroll analytics** to optimize budgeting and predict costs.

## Features
- **Employee Management:** Manage employee records, benefits, and payroll seamlessly.
- **Payroll & Benefits:** Automate payroll calculations, benefits allocation, and reporting.
- **Secure RESTful APIs:** Built with **Laravel Sanctum** and email-based PIN verification for authentication.
- **External System Integration:** Connects with finance, admin, and logistics systems via authenticated APIs.
- **AI Chatbot:** Real-time assistant powered by the **Gemini API** for employee queries and support.
- **Predictive Analytics:** AI-driven payroll analytics for budget visualization and cost prediction.

## Technologies Used
- **Backend:** Laravel, PHP
- **Frontend:** Blade Templates / Vue.js
- **Authentication:** Laravel Sanctum, Email-based PIN verification
- **AI Integration:** Gemini API
- **Database:** MySQL
- **External API Integration:** Finance, Admin, Logistics and other HR systems.
